# Changelog

## v0.1.0-alpha.1

- Initial repository scaffold and project setup.
- Implemented CLI (`qvac-bench`) for streaming latency benchmarking against a QVAC OpenAI‑compatible endpoint.
- Reports time to first token, total generation time, completion tokens, and approximate tokens/sec.
- JSON and CSV output formats (`--output json`, `--output csv`).
- Built‑in prompt fixtures: `hello`, `summary`, `reasoning`.
- Documentation: command‑line usage, methodology, and a report template (`docs/reports/template.md`).
